# Archived Documentation

This folder contains historical and reference documentation that is intentionally kept out of the active product-documentation path.

## Contents

- `customer_baseline/`: original intake and baseline customer documents.
- `starter_kit/`: extraction/starter-app guidance that is not required for current SelfSnap usage.
- `releases/`: historical standalone release notes.
- `implementation_legacy/`: superseded implementation planning and baseline-register documents.

Use `docs/README.md` for the active documentation index.
