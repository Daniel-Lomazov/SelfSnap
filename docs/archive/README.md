# Archived Documentation

This folder contains historical and reference documentation that is intentionally kept out of the active product-documentation path.

Nothing in this folder is the active end-user or maintainer contract for the current HEAD.
Use `docs/README.md` first when you need current guidance.

## Contents

- `customer_baseline/`: original customer intake pack, scope baseline, and handoff notes that shaped the later maintained docs. Start with `customer_baseline/README.md`.
- `starter_kit/`: archived starter-app extraction and reuse pack for future Windows tray utility work. Start with `starter_kit/README.md`.
- `releases/`: grouped historical release notes, rollout evidence, and retired release-specific validation, policy, trust, and internal behavior docs. Start with `releases/README.md`.
- `v1_0_planning_baseline/`: superseded implementation planning and baseline-register documents. Start with `v1_0_planning_baseline/README.md`.

Use `docs/README.md` for the active product and maintainer documentation index.
