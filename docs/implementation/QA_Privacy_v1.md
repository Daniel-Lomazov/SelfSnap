# SelfSnap Win11 v1 QA and Privacy Notes

## Privacy

- No cloud sync or remote upload
- No stealth mode
- Tray must remain visible while active
- Logs must not include image contents or inferred activity

## Manual QA focus

- Mixed monitor layouts
- Sleep through a schedule then reconcile
- Disabled and paused states
- Retention deletion and DB marking
- Packaging install and uninstall under a standard user account
